1. The sum of positive numbers(Easy)   
   • Problem: It receives a list of integers and returns the sum of the positive numbers.
   • Hint: Use filter to get the positive numbers and reduce to add them.

2. Length of words(Easy)   
   • Problem: It receives a list of words and returns a list with the length of each word.
   • Hint: Use map to apply the length function to each word.

3.Filtering even numbers
  • Problem: Receive a list of integers and return only the even numbers.
  • Hint: Use filter along with a lambda that checks if the number is even.

4.Application of a squaring function
  • Problem: It receives a list of numbers and returns a list of the square of each number.
  • Hint: Use map with a lambda that squares each number.

5.Capitalized word products
  • Problem: It receives a list of words and returns a list of those words that start with an uppercase letter.
  • Hint: Use filter with a lambda that checks if the first character is uppercase.

6. Convert letters to lowercase
  • Problem: It receives a list of words and returns a list of the same words converted to lowercase.
  • Hint: Use map with str.lower.

7. Multiplying all numbers
  • Problem: It receives a list of numbers and returns the product of all the numbers in the list.
  • Hint: Use reduce from functools.
